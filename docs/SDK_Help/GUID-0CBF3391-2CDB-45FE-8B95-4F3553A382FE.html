Network tab

Figure 34: Network tab

The Network tab is used for setting up emulator network settings. It offers the following http proxy setting options:

Option

Description

Use HTTP proxy

Used for defining whether to take HTTP proxy into use or not.

Proxy server name

Used for setting proxy server name

Proxy server port

Used for setting proxy server port

Restart the emulator for the changes to take effect.
