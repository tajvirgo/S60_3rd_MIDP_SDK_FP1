Supported features

In the S60 implementation, the Wireless Messaging API is supported as specified by the JSR-205 specification. Some of the classes are only partially supported. For information on these, see section Class and interface description.

The push feature (described in Appendix E of the JSR-205 specification) is supported in its entirety.

For more clarifications on the Java package level, see section Class and interface description, which describes the adopted implementation/solution of the specification where the specification might leave room for interpretations.

The S60 implementation also supports the GSM Cell Broadcast short message Service (CBS).
