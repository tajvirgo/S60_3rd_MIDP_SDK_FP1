vCard and vCalendar

The supported Versit personal information exchange formats are:

  • vCard 2.1 for Contacts

  • vCalendar 1.0 for Events and ToDos

The supported character encodings for the exchange formats are:

  • UTF-8 (default)

  • ISO-8859-1
