JAD/JAR checking

Real devices will perform MIDlet installation time checks to ensure consistency between the JAD file and the JAR file. Especially the manifest file in JAR needs to match with the JAD file as specified in the Mobile Information Device Profile (MIDP) Specification, Version 2.0. The S60 emulator does not perform these checks. Care must be taken to ensure, that JAD and JAR file contents are in line with the specification. Otherwise a MIDlet may run in the emulator but fail to install into a real device.
