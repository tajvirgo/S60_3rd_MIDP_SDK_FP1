Help contents

This Help provides information and instructions that you will need when working with the S60 Software Development Kit (SDK) to create new S60 applications with the Java™ programming language.

The Help is divided into the following main sections:

  • Introduction to SDK (current section) - general introduction to the S60 SDK and a Getting Started with Mobile Java section, aimed at beginners taking their first steps in S60 application development.

  • Tools and Utilities - information and instructions on the various tools provided with the SDK, including an S60 emulator guide.

  • Java Implementation Notes - information on the S60-specific implementation details of the APIs, including a list of supported and unsupported features as well as any class-level differences existing compared to reference implementation.

Note: Javadocs and S60 example MIDlet documentation are provided separately at the following locations in the SDK installation directory:

  • Javadocs: <S60_SDK_installation_directory>\docs

  • S60 example MIDlet documentation: <S60_SDK_installation_directory>\S60examples
