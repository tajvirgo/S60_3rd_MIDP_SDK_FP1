Runtime environment notes

System properties

The Web Services API (JSR-172) has the following system properties:

String

Description

xml.jaxp.subset.version

The version of JAXP Subset APIs supported by the device. The returned value is 1.0.

xml.rpc.subset.version

The version of JAXP-RPC Subset APIs supported by the device. The returned value is 1.0.

Security settings

JSR-172 does not define any security settings. Security features from MIDP 2.0 security framework will apply.
