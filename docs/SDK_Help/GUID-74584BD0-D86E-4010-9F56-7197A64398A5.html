Client startup

Follow these steps to start up the ECMT client:

  1. From the Start menu, select Start > Programs > S60 Developer Tools > 3rd Edition SDK > MIDP > Tools > Device Connection.

  2. The Device Connectivity Tool for S60 SDK window is displayed:

    Figure 81: Device Connectivity Tool for S60 SDK - not connected

  3. From the Port list, select the COM port which your PC's Bluetooth adapter has been configured to use for serial port connections.

  4. Select File > Connect.

    Figure 82: Device Connectivity Tool for S60 SDK - connected

After a connection between the PC and the phone has been established, select Tools > Diagnostics to open the Diagnostics window.
