Saving the location file

When you are finished with defining the route, save it in the desired format (for example, click Save and enter a filename with the suffix .nme file extension) and then, click the Apply button in the SimPSY Route dialog .

Clicking Apply communicates the updates for the emulator and configures SimPSY to use the newly created file for location information. Thus, you do not have to manually instruct SimPSY to load the new location file, you can directly start fetching the coordinates. When saving the location file it is important that you use the .nme file extension for NMEA files and the .sps file extension for SPS files, as the SimPSY Configurator displays files of these types only, and thus does the route plug-in.
