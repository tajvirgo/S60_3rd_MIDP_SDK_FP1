Package support

All the packages defined in the JSR-75 specification are supported according to the table below:

Package

Supported classes/interfaces

Unsupported classes/interfaces

javax.microedition.io

Connector

This is a generally scoped package. Only the Connector class of this package is targeted by File API and it is supported. The other classes (if any) of this package are not relevant to File API scope and therefore they are not supported.
