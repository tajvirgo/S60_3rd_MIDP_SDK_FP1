Stopping the SIP Server Emulator

Context

Once you stop the emulator, all outgoing traffic stops immediately and no new SIP messages are forwarded.

Steps

  1. Select Server > Stop server .

Alternative steps

  1. Click the Stop button.

Alternative steps

  1. You can also shut down the SIP Server Emulator by selecting Server > Exit or closing the SIP Server Emulator UI window.
