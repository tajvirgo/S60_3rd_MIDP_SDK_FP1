From S60 3rd Edition to S60 3rd Edition FP 1

New standard fields for ContactList:

  • Contact.NICKNAME

  • Contact.REVISION

  • Contact.CLASS

New extended fields for ContactList:

  • SIP (Session Initiation Protocol)

  • DTMF (Dual-Tone Multi-Frequency)

  • USER_ID (Wireless Village identifier)

  • Department

  • Spouse

  • Children

  • Assistant name

  • Anniversary

  • VOIP (Voice Over IP)

New Contact.NAME name array elements:

  • Contact.NAME_PREFIX

  • Contact.NAME_SUFFIX

  • Contact.NAME_OTHER

New extended Contact.NAME name array elements:

  • Last name reading

  • First name reading

New attributes for Contact.TEL field:

  • Contact.ATTR_ASST

  • Contact.ATTR_AUTO

  • Contact.ATTR_SMS (preferred SMS number)

  • Contact.ATTR_OTHER

  • Extended attribute ATTR_VIDEO_CALL

New fields for EventLists:

  • Event.CLASS for all EventLists

  • Event.NOTE for all EventLists

  • Event.REVISION for all EventLists

New fields for ToDoList:

  • ToDo.CLASS

  • ToDo.REVISION

  • ToDo.NOTE

  • Extended Alarm field

ToDos without ToDo.SUMMARY now accepted

Additional confirmation dialogues are not shown in the Manufacturer security domain when an item is written to the database or when it is deleted from the database. These dialogues are also not shown when a category is to be deleted.
