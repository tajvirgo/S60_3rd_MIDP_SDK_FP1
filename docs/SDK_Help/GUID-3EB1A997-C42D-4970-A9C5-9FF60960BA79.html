Symbian Developer network

The Symbian Developer network provides downloads, technical papers, system documentation as well as more interactive elements such as newsletters and discussion forums.

  • Tools and SDKs - Developer tools, utility libraries and API extensions, example applications, and open source projects

  • Library - Symbian OS documentation and technical papers.

  • Academy - Symbian Press, Symbian Academy, accredited Symbian developer, FAQs, training courses.

  • Symbian Signed - Symbian Signed is an industry-backed program that allows you to certify your Symbian OS applications so that they can be installed and used on S60 3rd Edition and UIQ 3 phones. Certifying an application via Symbian Signed can greatly improve the application's route and time to market.

  • Forums - Discussion groups and feedback areas.
