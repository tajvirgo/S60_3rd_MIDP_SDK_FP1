Runtime environment notes

System properties

Name of system property

Description of purpose

microedition.amms.version

The String returned designates the version of the AMMS specification that is implemented.

supports.mediacapabilities

The String returned specifies the supported capabilities, delimited by one whitespace (Unicode U+0020).

audio3d.simultaneouslocations

The String returned specifies the maximum number of SoundSource3D modules that the application is recommended to have active simultaneously in a typical game playing situation when graphics rendering (2D or 3D) and reverberation are also taking place.

audio.samplerates

The String returned specifies the recommended audio sampling rates in Hertzs, delimited by one whitespace (Unicode U+0020).
