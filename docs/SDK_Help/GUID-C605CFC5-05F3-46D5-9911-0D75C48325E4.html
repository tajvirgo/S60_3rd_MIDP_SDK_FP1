Package support

All the packages defined in the SIP API for J2ME (JSR-180) specification are supported by the S60 implementation according to the table below:

Table 27: Supported packages

Package

Supported classes/interfaces

Unsupported classes/interfaces

javax.microedition.io

Connector

This is a generally scoped package. Only the Connector class of this package is targeted by SIP API for J2ME (JSR-180) and it is supported.The other classes (if any) of this package are not relevant to the scope of SIP API for J2ME (JSR-180) and their support is not discussed in this document.
