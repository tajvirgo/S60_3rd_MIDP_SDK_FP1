Package support

All the packages defined in the JSR-205 API specification are supported according to the table below:

Package

Supported classes/interfaces

Note

javax.microedition.io

Connector

This is a general-scope package. Only the Connector class of this package is targeted by JSR-205 and it is supported. The other classes (if any) of this package are not relevant to the JSR-205 scope and are therefore not supported.

javax.wireless.messaging

Full support
