Starting the S60 emulator from the command line

Steps

  1. Use the CD command and then enter the SDK installation directory into the Command Prompt:

    CD C:\<SDK_installation_directory>\bin\epoc32\release\winscw\udeb

  2. To this command add: sdk.exe. The complete command should look like this:

    C:\<SDK_installation_directory>\bin\epoc32\release\winscw\udeb>sdk.exe
