1. About this example

This is the general portion of the Bluetooth SPP example.

2. Pre-requisites

The example makes use of the standard MIDP application framework and LCDUI API. The reader should be familiar with these areas before attempting to understand this example.

3. Building and Running

The example is built using the standard method for building and running as described in Building and running the examples.

The following classes should be included in the jar:

com.nokia.midp.examples.btsppEcho.*

No resource files are used in this example.

For more information on running the midlet see the client and server package level documentation.

4. Design and implementation

For more detailed information see the client and server package level documentation.

5. Summary

This example illustrates the implementation of a midlet that makes use of Bluetooth/SPP.
