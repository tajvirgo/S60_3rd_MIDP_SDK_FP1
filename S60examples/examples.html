S60 SDK Example MIDlets

This is the starting point for accessing the documentation for example MIDlets delivered with the S60 3rd Edition SDK for Symbian OS Supporting Feature Pack 1, for MIDP. The example MIDlets demonstrate the variety of different applications that you can run on S60 devices.

You can build and run these applications on the S60 emulator. The MIDlets can also be used in your own application development. Each example has a set of project files for building from the command line and various integrated development environments.

The S60 example MIDlets are located by default in the following folder after SDK installation:

<S60_SDK_installation_directory>\S60examples

Each example MIDlet is provided with its own Java™ documentation. The documentation can be found in the following location:

<S60_SDK_installation_directory>\S60examples\<example_MIDlet>\doc\javadoc\index.html

The example MIDlets are also provided with general S60 Example MIDlet Help documentation. This help documentation set provides a comprehensive list of all example MIDlets delivered with the SDK as well as instructions on how to build and run the MIDlets from supported IDEs and command line interface. The S60 Example MIDlet Help is located here:

<S60_SDK_installation_directory>\S60examples\doc-files\index.html
