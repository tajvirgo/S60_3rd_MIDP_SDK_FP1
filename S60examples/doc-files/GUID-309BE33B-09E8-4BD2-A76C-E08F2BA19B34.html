Viewing example MIDlets in the S60 emulator

Context

To open an example MIDlet in the S60 emulator, do the following:

Steps

  1. Open the S60 emulator
  2. Select File > Open... from the emulator menu bar.
  3. In the open dialog, browse to the bin directory of the example MIDlet that you want to view in the emulator.

    For example, <S60_SDK_installation_directory>\S60examples\helloworldplus\bin.

  4. Select the .jar file of the example MIDlet and click Open.

    For example, helloworldplus.jar.

  5. When prompted to confirm installation of the MIDlet to the emulator, click left softkey for Yes.
  6. If the emulator issues a security warning, click the left softkey to Continue the installation.
  7. When the MIDlet details are displayed by the emulator, click the left softkey to Continue the installation.
  8. Select the location for the example MIDlet (phone memory or memory card) and click the left softkey to Select.

Results

The MIDlet is installed to the emulator. To open and view the MIDlet, do the following:

  1. Click the Applications button to open the application grid.

  2. In the application grid, use the five-way navigation key to go to the Installed folder.

  3. Open the Installed folder by:

    • clicking the centre of the five-way navigation key, or

    • select Options > Open with the left softkey.

  4. In the Installed folder, use the five-way navigation key to locate the application icon of the MIDlet that you have installed.

  5. Open the MIDlet by:

    • clicking the centre of the five-way navigation key, or

    • select Options > Open with the left softkey.
