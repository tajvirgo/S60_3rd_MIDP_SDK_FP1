Building overview

This document explains the process required to build and run the various examples. If an example has additional building and running requirements or dependencies, these are detailed in the Building and running section for that particular example.

Source code for the examples is located in

<SDK_installation_directory>/<examples folder>

A syntax-highlighted version of the source is also included, this is linked to from the relevant class in the JavaDoc documentation.

Pre-requisites

Before you start, you should have the MIDP SDK, the Java™ Platform Standard Edition SDK and, optionally, an IDE tool. If you use an IDE, you should also first configure the emulator for use with the IDE tool. For more details, see the Installation and Configuration Guide included as a separate pdf document in the SDK.

Common terminology

The details in this section apply to all of the tools.

root directory of the source

 This is the directory containing a subdirectory called "com". It is not the "com" directory itself, or the directory containing .java files.

Tools

This section gives download locations and brief notes for the main tools, which are straightforward download-and-run installations. The Java Platform Standard Edition SDK must be installed.

Java™ Platform Standard Edition (SDK)

Downloads : http://java.sun.com

The Java Platform Standard Edition SDK provides the basic tools for compiling and running Java programs.
