Introduction

The S60 3rd Edition SDK for Symbian OS, Supporting Feature Pack 1, for MIDP includes a number of example MIDlets, which demonstrate the variety of different applications that you can run on S60 devices. You can build and run these applications as well as use them in creating your own applications. The MIDlets are supplied with Java™ documentation including design diagrams that use UML Notation. Each MIDlet has a set of project files for building from the command line and various integrated development environments (IDEs).

The MIDlets are located by default in the following folder after SDK installation: <S60_SDK_installation_directory>\S60examples. For a list of example MIDlets provided with the SDK, please refer to S60 example MIDlets.

For instructions on how to run the example MIDlets on the S60 emulator, please refer to:

  • Building and running example MIDlets from the command line

  • Running an S60 SDK example MIDlet from Carbide.j

  • Running an S60 SDK example application from NetBeans

  • Launching an S60 example MIDlet from WSDD

The UML notation primer provides a brief overview of UML class diagrams and sequence diagrams used in the example documentation
