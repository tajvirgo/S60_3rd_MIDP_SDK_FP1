Introduction

The following instructions introduce you using the NetBeans development and design tools with the MIDlet examples..

Follow the installation instructions for provided in the NetBeans documentation. 

Prerequisites

Before you can specify the SDK to be used in a project, you have to configure it to be used with NetBeans. This involves adding it to the set of available JDKs. For more detailed instructions, see the J2ME MIDP Development Quick Start Guide provided by NetBeans.

Opening the project   

Open an example MIDlet NetBeans project file by using the File -> Open Project selection from the NetBeans main screen.. The NetBeans project file is found within the targets subdirectory for the example.  For example the NetBeans project files for the helloworldplus example are found in the directory <examples folder>/helloworldplus/targets/netbeans folder.  To open the project use the NetBeans file menu open project option.   A file selector window will appear.  Navigate to the targets subdirectory of the example you want to use and select the project file as shown below.


Configuring the JDK   

Use the instructions in the J2ME MIDP Development Quick Start Guide Project menu Project Properties selection to configure the JDK (The SDK you use for the midlet).

 

Building and running the midlet

  1. Rebuild the project by selecting, for example, Build > Build Main Project from the Main menu.

  2. Select Run > Run Main Project from the Main menu.

The emulator opens displaying, first, the Applications content view. Then, the MIDlet is launched and message "Hello World" appears on the screen.

Once you have tested the application and you are happy with it, you can install the application to your smartphone. As the installation procedures are device-specific, see the installation instructions provided for your smartphone model.
